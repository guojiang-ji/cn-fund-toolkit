# 中国基金实用工具包

为国内基金投资者（A股/港股/美股QDII）提供三个核心工具：持仓分析、基金对比、定投计算器。

## 为什么需要这个技能

现有财经AI工具的问题：
- 数据来自Yahoo Finance，对A股基金支持差
- 术语英文化，不适合国内基民阅读习惯
- 没有针对定投场景的复利计算

这个技能用国内公开数据源，输出中文基民熟悉的格式。

## 三大工具

### 1. 持仓分析器

**输入：** 你的基金持仓列表（名称+金额+买入日期）
**输出：** 持仓结构分析 + 风险提示

**分析维度：**
- 行业集中度（科技/医药/消费/新能源占比）
- 风格暴露（大盘/小盘/价值/成长）
- 地域分布（A股/港股/美股）
- 风险热力图（建议最大回撤容忍度）

**示例：**
```
输入：
  银河创新成长混合C 5000元 2024-03-01
  农银专精特新混合C 8000元 2024-05-15
  富国恒生科技指数C 3000元 2025-06-10

输出：
  ┌─────────────────────────────────────┐
  │ 持仓结构分析                        │
  │                                     │
  │ 科技成长型：71% ⚠️ 集中度偏高       │
  │ 大盘蓝筹：0%   ❌ 缺失防御板块    │
  │ 港股/美股：21%  ✓ 分散合理        │
  │                                     │
  │ 建议：                               │
  │ 1. 添加一只债券或宽基指数作为底仓   │
  │ 2. 科技占比超过70%，需关注回撤风险  │
  │ 3. 港股QDII刚买入，注意汇率波动    │
  └─────────────────────────────────────┘
```

### 2. 基金对比器

**输入：** 2-5只基金名称
**输出：** 多维度对比表格

**对比维度：**
- 业绩表现（近1月/3月/6月/1年/3年）
- 风险指标（最大回撤、夏普比率、波动率）
- 费用结构（管理费、托管费、申购赎回费）
- 基金经理（任期、历史业绩、管理规模）
- 持仓风格（行业集中度、前十大重仓）

### 3. 定投计算器

**输入：** 基金代码、定投金额、定投频率、目标收益率
**输出：** 预期收益曲线 + 回本时间估算

**计算逻辑：**
- 使用历史净值数据模拟定投
- 考虑申购费率和分红再投资
- 输出乐观/中性/悲观三种情景

**示例：**
```
输入：定投银河创新成长混合C，每月1000元，目标年化15%

输出：
  基于近3年数据回测：
  - 乐观情景（年化20%）：3年累计 ≈ 46,000元，投入36,000元
  - 中性情景（年化12%）：3年累计 ≈ 42,000元
  - 悲观情景（年化5%）：3年累计 ≈ 38,000元
  - 回本时间：约14-18个月（历史回测）
```

## 数据来源

- 基金基础信息：天天基金网公开API
- 净值数据：新浪财经基金接口
- 基金经理信息：好买基金公开数据
- 所有数据均为公开信息，无需账户登录

## 使用方式

**持仓分析：**
```
你：帮我分析一下持仓
我：请提供你的基金列表（名称 金额 买入日期）
你：银河创新成长混合C 5000元 2024-03-01，农银专精特新混合C 8000元...
我：[直接输出分析结果]
```

**基金对比：**
```
你：银河创新和农银专精特新哪个好？
我：[直接输出对比表格]
```

**定投计算：**
```
你：每月定投1000元银河创新，3年后大概多少？
我：[直接输出计算结果]
```

## 技术细节

- **无需API Key：** 使用公开网页数据抓取，无需付费
- **Python脚本：** 本地计算，不依赖外部LLM，速度快
- **缓存机制：** 基金基础数据每日更新，净值数据实时

## 注意事项

⚠️ 本工具仅供学习参考，不构成投资建议
⚠️ 历史业绩不代表未来表现
⚠️ 基金数据来自公开接口，可能存在延迟
